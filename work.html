Work

Projects from software, product work, and creative tools.

I like building useful things with personality, from production-ready apps to playful interactive experiences.

Stack

Tools I have used across these projects.

My public projects lean toward TypeScript and JavaScript, with a mix of browser extensions, hand-built interfaces, API experiments, Go utilities, and small scripts.

I tend to use the stack that fits the problem: a browser extension when the work needs to happen where someone already is, a small API when the logic needs to be reusable, a static site when speed and clarity matter, and a script when the best interface is just a command that saves time.

Chord Function API
