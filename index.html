Chord Function API

Problem: Harmonic analysis is powerful, but manually applying function logic is slow and inconsistent in apps.

Approach: I designed an API around key center, root, and chord quality to make functional harmony queryable.

Outcome: It enables reliable chord-function output that can plug directly into music tools and creative workflows.
